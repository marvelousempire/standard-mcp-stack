# Knowledge Source Standard

Knowledge sources are governed evidence repositories that the system can consult. They are separate from memory and separate from model weights.

## Trust lens

Knowledge sources are deeds, certificates, manuals, correspondence, contracts, archives, and authoritative records.

## Business lens

Knowledge sources are repositories, documentation sites, databases, wikis, websites, and shared drives.

## Source classes

- local files
- NAS shares
- Git repositories
- Obsidian vaults
- WordPress content
- relational databases
- vector databases
- websites and APIs
- document-management systems

## Source registry fields

- source ID
- source type
- location
- owner
- sensitivity
- authority level
- allowed operations
- indexing policy
- freshness policy
- citation method
- retention policy

## Provenance rule

Every material answer grounded in knowledge should retain enough information to identify the source, retrieval time, and relevant location.
